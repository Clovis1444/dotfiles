* About

This is my dotfiles for *Arch* - *Niri* - *DankMaterialShell* setup.

* Install

#+begin_src bash
cd $HOME
git clone https://github.com/Clovis1444/dotfiles.git
cd ./dotfiles
./configure-gitconfig.sh
stow .
#+end_src

#+begin_quote
[!IMPORTANT]
For installing dependencies see [[system-setup.sh][system-setup.sh]].
#+end_quote

* Usage

This repo uses [[https://www.gnu.org/software/stow/][Stow]] for managing dotfiles.

Do all dotfiles changes inside =~/dotfiles/=.

To add new config to the repo do:
#+begin_src bash
mv ~/.config/nvim ~/dotfiles/.config/nvim
cd ~/dotfiles && stow .
#+end_src

#+begin_quote
[!TIP]
To make changes into ~.gitconfig~ edit [[.gitconfig.in][~/dotfiles/.gitconfig.in]], then run [[configure-gitconfig.sh][configure-gitconfig.sh]].
#+end_quote
